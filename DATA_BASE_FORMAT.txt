Любой файл базы данных Meow начинается со строки метаданных.
{ KEY: (HASH_CODE|NULL), FORMAT: [ TYPE: [NAME], TYPE: [NAME], ... , TYPE: [NAME] ] }

KEY - поле отвечающее за защиту файла. Если значением KEY является некий HASH_CODE, то 
файл будет доступен при вводе пароля. Если NULL, то файл открыт для всех.

FORMAT - поле отвечающее за то, как будут выглядеть поля данных.

Пример:
    #Метаданные
    { KEY: NULL, FORMAT: [String: Name, Integer: Age] }
    #Данные метаданные означают, что файл открыт для всех, и что первый столбец данных
    #это строка содержащая некоторое имя, а второй столбец целое число - количество лет.

Исходя из этих метаданных поля данных будут выглядеть так:
    [$N, "Petya", 24]
    ...
    [$M, "Ann", 19]

Что же такое $N и $M?
    В поля данных по умолчанию записывается их номера. Данная запись $N - говорит интерпретатору запросов,
    что N - это номер поля.
    Поэтому пятое поле выглядит примерно так:
        [$5, "Hans", 56]
